💡 Troubleshooting
Don't see View/Edit/Share buttons?
→ Hard refresh browser: Ctrl+Shift+R
→ Check browser console for errors: F12
→ Verify you're logged in with admin/editor role

Hotspot editor not showing?
→ Make sure you clicked "Edit" mode first
→ Go to Hotspots tab in sidebar
→ If in View mode, you'll see "Enter Edit Mode" button

Autosave not working?
→ Must be in Edit mode
→ Must select an existing hotspot
→ Check browser console for errors
